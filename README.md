# Eraser

An ESP32-S3 based frequency analyzer that supports NRF24/CC1101 and Custom GPIO Pins (Like Flipper Zero). This project uses an 1.9 inch ST7789 TFT, with 5 buttons (maybe rotary encoder in sooner version) and Infared TX and RX.

To configure your pins, please edit config.h in /include
